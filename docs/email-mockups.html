ChimpFlix — Email mockups

Design review for every email the server can send. The wrapper around each preview mimics a Gmail-style inbox row so the subject line + sender are evaluated alongside the body itself. The email bodies themselves use inline-style-equivalent CSS so porting back to the Rust template builders is a 1:1 translation of selectors → inline style="" attributes.
